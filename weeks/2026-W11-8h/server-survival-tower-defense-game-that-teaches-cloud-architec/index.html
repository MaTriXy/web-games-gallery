    Controls

    • Move tower: Swipe left or right -> Shift the server tower between the three lanes.
    • Swap stack color: Tap -> Cycle the tower between red, blue, and yellow service modes to match the next gate.
    • Autoscale burst: Tap and hold during a spike alarm -> Deploy a temporary sidecar clone into an adjacent lane for a few seconds, catching one lane automatically while you keep
    • Recover after hit: Release and keep moving -> Return to single-tower control and rebuild your combo before the next surge.

    Plan

    1. Match the tower's color to the next request gate in its lane before contact.
    2. Each correct match clears traffic, raises combo, and slightly increases game speed.
    3. A wrong color or wrong lane lets bad traffic through and removes one stability segment.

    Pilot tips

    • Read two gates ahead; lane choice matters as much as color choice.
    • Use autoscale only on double-lane spike waves, because it protects survival but weakens combo gain.
    • Audio pings can cue color families and overload alarms, making the game playable offline without constant visual focus.
