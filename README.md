# Matrixy Web Games

Static game gallery exported from `web_games_agent`.

## Contents

- `index.html`: gallery landing page
- `catalog.json`: machine-readable list of exported weeks and games
- `weeks/<weekKey>/...`: copied weekly build artifacts and playable game bundles

1. Push this repository to GitHub.
2. In repository settings, enable Pages for the default branch root.
3. Open the published `index.html` site URL.
